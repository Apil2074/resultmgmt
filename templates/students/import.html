Instructions

Please format your Excel spreadsheet using the standardized template to prevent enrollment errors.

Required Columns & Format:
  • Roll No* — Roll number (unique per class)
  • Symbol No — Board exam symbol number (optional)
  • REG NO — Registration number (optional)
  • Name of Students* — Full student name
  • Gender — Male / Female / Other (optional)
  • Date of Birth BS. — Date of birth in BS or standard format (optional)
  • Parent Name — Father/Mother/Guardian name (optional)
  • Contact Number — Phone number (optional)
  • Address — Home address (optional)
Note: The class is determined by the Excel Sheet Name (e.g., a sheet named "Class 10 A" imports students into that class). You can import multiple classes at once using multiple sheets.
